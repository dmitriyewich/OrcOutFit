# Настройки

Страница описывает вкладку `Settings`.

## Назначение вкладки

- Управление способом открытия меню.
- UI/языковые параметры.
- Прочие пользовательские настройки, не привязанные к отдельному функциональному блоку.

## Ключевые разделы

### Активация меню

- `ActivationKey` для SP (и опционально SA:MP, если разрешено).
- Команда для SA:MP.
- Разрешение клавиши в SA:MP настраивается отдельно.

### Язык

- Переключение локализации интерфейса (`ru`/`en`).

### Интерфейс

- Масштаб UI.
- Размер шрифта.
- Дополнительные параметры читаемости/удобства.

## Пошаговая инструкция по базовой настройке

1. Настрой способ открытия меню под свой режим игры.
2. Выбери язык интерфейса.
3. Подстрой масштаб/шрифт под разрешение.
4. Перезапусти игру, если конкретная настройка этого требует.

## Пошагово: рекомендуемая настройка для SP

1. Установи удобную `ActivationKey`.
2. Проверь открытие/закрытие меню в игре несколько раз подряд.
3. Отрегулируй `UiScale` и `UiFontSize` под своё разрешение.
4. Сохрани параметры.

## Пошагово: рекомендуемая настройка для SA:MP

1. Установи и проверь чат-команду для открытия меню.
2. При необходимости отдельно разреши открытие по клавише.
3. Проверь, что команда стабильно работает в твоем клиенте.
4. Если есть конфликты с биндами сервера, выбери другую команду/клавишу.

## Практические рекомендации

- Для SA:MP держи в приоритете команду, а не только клавишу.
- При проблемах с отображением сначала верни дефолтные UI-значения.
- После крупного изменения UI-параметров перезапусти игру, если видишь артефакты интерфейса.

## Частые проблемы и решения

### Клавиша не работает

- Проверь корректность virtual key и отсутствие конфликта с другими модами.
- В SA:MP проверь, разрешено ли открытие по клавише.

### Команда не работает в SA:MP

- Убедись, что команда начинается с `/`.
- Проверь, не занята ли команда сервером или другим модом.

### UI слишком мелкий или крупный

- Верни `UiScale` ближе к 1.0.
- Подними/снизь `UiFontSize` небольшими шагами.
