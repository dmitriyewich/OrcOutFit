# OrcOutFit Wiki (локальные черновики)

Этот каталог хранит локальные черновики страниц wiki, чтобы можно было спокойно редактировать их в репозитории до публикации.

## Страницы

- `Описание-проекта.md`
- `Главная.md`
- `Оружие.md`
- `Объекты.md`
- `Скины.md`
- `Настройки.md`

## Как поддерживать

- Держи эти страницы синхронизированными с фактическим поведением плагина.
- При заметных изменениях UI или логики сначала обновляй локальные страницы, затем публикуй в GitHub Wiki по готовности.
