# Объекты

Страница описывает вкладку `Objects` и её подвкладки.

## Структура вкладки

- `Custom objects`
- `Standard objects`

---

## Custom objects

### Что делает

- Подключает кастомные DFF-объекты из папки `OrcOutFit\Objects`.
- Позволяет задавать параметры крепления отдельно для разных стандартных ped-скинов.

### Пошаговая инструкция

1. Положи нужный `*.dff` в `OrcOutFit\Objects`.
2. Создай/проверь INI-файл объекта (`<имя>.ini`) рядом с DFF.
3. В INI настрой секции вида `[Skin.<dff_name>]` для нужных стандартных ped.
4. Включи отображение кастомных объектов во вкладке `Objects`.
5. Выполни рескан объектов в UI.
6. Проверь отображение объекта на нужном ped в игре.

### Что важно проверить

- Имя `<dff_name>` в секции должно совпадать с базовым ped DFF.
- Если объект не виден, сначала проверь `Enabled`, кость и смещения.
- При редактировании INI на лету выполняй рескан, чтобы изменения подхватились.

## Standard objects

### Что делает

- Добавляет стандартные игровые модели по model ID в список рендера.
- Позволяет использовать несколько слотов для одного model ID (`123#1`, `123#2` и т.д.).

### Пошаговая инструкция

1. Открой `Objects -> Standard objects`.
2. Введи model ID и добавь его в список.
3. Выбери нужный слот (`ID#N`), если ID повторяется.
4. Выбери ped skin, для которого настраиваешь объект.
5. Настрой кость, offset, rotation, scale и ограничения по оружию (если нужны).
6. Сохрани настройки.
7. Проверь результат в игре и при необходимости скорректируй параметры.

### Что важно проверить

- Используй только валидные object model ID (ped/vehicle ID не подходят).
- Если объект не добавляется, модель может отсутствовать в текущей сборке игры/SA:MP.
- Для нескольких копий одного объекта используй отдельные слоты, а не перезапись одного.

## Полезные сценарии

### Сценарий 1: один кастомный объект для конкретного скина

1. Добавь DFF в `OrcOutFit\Objects`.
2. В `<имя>.ini` создай `[Skin.<dff_name>]`.
3. Настрой положение и включи `Enabled`.
4. Выполни рескан и проверь в игре.

### Сценарий 2: два одинаковых стандартных объекта в разных позициях

1. Добавь один и тот же model ID дважды.
2. Получи два слота (`ID#1`, `ID#2`).
3. Для каждого слота отдельно настрой кость и трансформации.
4. Сохрани и протестируй в движении.

### Сценарий 3: объекты у всех ped в радиусе

1. Включи режим рендера объектов для всех ped (если используешь этот режим).
2. Установи разумный радиус.
3. Проверь производительность в плотных сценах.
