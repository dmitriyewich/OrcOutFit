# Скины

Страница описывает вкладку `Skins` и её подвкладки.

## Структура вкладки

- `Custom skins`
- `Standard skins`
- `Random skins`
- `Texture`

---

## Custom skins

### Что делает

- Рендерит кастомный DFF-скин поверх текущего ped.

### Пошаговая инструкция

1. Положи DFF (и при необходимости TXD) в `OrcOutFit\Skins`.
2. Выбери скин в `Custom skins`.
3. Настрой дополнительные опции (например, скрытие базового ped).
4. Сохрани настройки и проверь результат в игре.

### Важно

- Для SA:MP при активном nick binding приоритет может быть у скина по нику.
- Если оверлей не виден, проверь совместимость модели и корректность файлов.

---

## Standard skins

### Что делает

- Использует стандартные игровые ped-модели для overlay и примерки локального игрока.

### Пошаговая инструкция

1. Выбери стандартный скин из списка.
2. Примени как overlay или через `Wear this skin (local player)`.
3. При необходимости настрой nick binding.
4. Проверь поведение после смены оружия/анимаций.

### Важно

- Для локального игрока учитывай параметры приоритета выбранного скина и скина по нику.
- Если нужного ped нет в списке, проверь, существует ли он в текущей сборке игры/клиента.

---

## Random skins

### Что делает

- Назначает random-варианты из пулов `OrcOutFit\Skins\Random\<dff>\*.dff`.

### Пошаговая инструкция

1. Подготовь папки random-пулов.
2. Включи режим random.
3. Выполни рескан и проверь распределение вариантов.
4. Убедись, что имя папки совпадает с базовым DFF ped.

### Важно

- Random-пулы работают только при корректной структуре папок.
- Выбор варианта может кешироваться и не меняться каждый кадр (это нормальное поведение).

---

## Texture

### Что делает

- Подменяет текстуры стандартных ped-моделей через remap-логику.

### Пошаговая инструкция

1. Включи texture remap.
2. Настрой random/nick/auto-режимы по необходимости.
3. Проверь результат на локальном ped и на других ped.
4. Если используешь ник-режим, проверь соответствие имени в SA:MP.

## Важные ограничения

- Длинные имена в texture binding могут упираться в лимиты RenderWare.
- Логика приоритетов (nick -> auto -> random) должна быть явно описана и проверяема.

## Полезные сценарии

### Сценарий 1: всегда использовать выбранный скин на себе

1. Выбери скин в `Custom skins` или `Standard skins`.
2. Включи опцию приоритета выбранного скина для локального игрока.
3. Проверь, что nick binding не переопределяет выбор.

### Сценарий 2: разные скины для игроков по никам

1. Включи режим nick binding.
2. Настрой список ников в соответствующем INI.
3. Проверь в SA:MP, что ник распознается корректно.

### Сценарий 3: random-скины только для определенных базовых ped

1. Подготовь пулы только для нужных `<dff>`.
2. Включи random-режим.
3. Проверь, что ped без пула остаются без random overlay.
